<Note>StarSearch requires a GitHub login to use.</Note>

StarSearch is an AI-powered assistant built into contributor.info. Open it from any repository page and ask questions about contributors, PR activity, health, and recommendations — no dashboards to navigate.

## How to use it

1. Navigate to any repository on contributor.info
2. Click the **StarSearch** button in the bottom-right corner
3. Type a question or pick a suggested prompt

## Example questions

- "Give me an overview of this repo"
- "Which PRs need attention right now?"
- "How healthy is this repository?"
- "What recommendations do you have?"
- "Who are the top contributors?"

StarSearch answers using the same 30-day contribution data that powers the rest of contributor.info — lottery factor, PR metrics, contributor rankings, and health scores.

## What it replaces

StarSearch replaces the previous Insights pages (PR Activity, Repository Health, Needs Attention, Recommendations). Instead of navigating between separate dashboards, ask StarSearch directly and get a focused answer.

## Status

StarSearch is a work in progress — more capabilities are coming soon.
