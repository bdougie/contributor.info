## Quick Start

1. Visit [contributor.info](https://contributor.info)
2. Search for your GitHub repository
3. View instant insights about your contributors
4. Sign in for advanced features and tracking

## Key Features

- **Lottery Factor**: Understand repository risk and contribution centralization
- **Activity Feed**: Track real-time contributor and repository activity
- **Distribution Charts**: Visualize contribution patterns and trends
- **Repository Health**: Comprehensive health metrics for your repository
- **Contributor Confidence**: Measure how welcoming your repository is
