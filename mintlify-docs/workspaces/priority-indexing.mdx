---
title: Priority Indexing
description: Skip the daily processing queue and get your repository data updated immediately.
---

<Note>Priority indexing is a paid feature available with Workspaces.</Note>

Large repositories are queued for daily processing by default. Priority indexing moves your repository to the front of the queue so data is available faster.

## When to use it

- You just added a large repository and want data immediately
- You need up-to-the-minute metrics for a meeting or review
- A repository had a burst of activity and you want the dashboard to reflect it now

## How to request

1. Navigate to the repository page
2. Click **Request Priority** in the banner at the top of the Activity tab
3. Data processing begins within minutes

Priority indexing is also available through [GitHub Discussions](https://github.com/bdougie/contributor.info/discussions).
