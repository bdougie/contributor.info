<Note>Repository tracking is a paid feature available with Workspaces.</Note>

When you add a repository to a workspace, contributor.info begins tracking it automatically. New PR activity, contributor changes, and health metric shifts are captured and surfaced in your workspace dashboard.

## What gets tracked

- **New pull requests** — opens, merges, and closes
- **Contributor activity** — new contributors, returning contributors, contributor churn
- **Health metric changes** — lottery factor shifts, confidence score changes, merge time trends
- **Discussion and issue activity** — new threads and responses needing attention

## How it works

1. Add a repository to your workspace
2. contributor.info syncs the last 30 days of data immediately
3. Ongoing activity is tracked automatically — no polling or manual refresh needed
4. Changes appear in the workspace **My Work** feed and **Activity** tab

## Data freshness

Tracked repositories are updated daily. For faster updates, use [Priority Indexing](/workspaces/priority-indexing).
