<Note>Slack integration is a paid feature available with Workspaces.</Note>

Connect a Slack channel to your workspace to receive notifications about repository activity without leaving your team's communication tool.

## What gets posted

- New PR activity summaries
- Contributor milestones (first PR, contributor of the month)
- Health alert changes (lottery factor shifts, merge time spikes)

## Setup

1. Open your workspace **Settings** tab
2. Click **Connect Slack**
3. Authorize the contributor.info Slack app
4. Choose which channel receives notifications

## Customization

You can configure notification frequency and which event types are posted from the workspace Settings tab.
